## 📖 Deskripsi Singkat  
**App-Antrian** adalah aplikasi antrian digital yang dikembangkan untuk **Direktorat Metrologi**, dirancang untuk meningkatkan efisiensi layanan publik, mempermudah pemanggilan nomor antrian, serta menampilkan informasi secara real-time melalui tampilan display.

Aplikasi ini dibangun dengan pendekatan modern, stabil, dan mudah dioperasikan oleh petugas pelayanan.

---

## 🧩 Struktur Sistem Aplikasi  

Aplikasi terdiri dari **tiga komponen utama** yang saling terhubung:

### **1. 🎟️ Kiosk App**  
- Bertindak sebagai **server utama** dan pusat database  
- Tempat pengunjung mengambil nomor antrian  
- Mengirim data secara real-time ke Admin App & Display  

---

### **2. 🖥️ Admin App (Operator Loket)**  
Digunakan oleh petugas untuk mengelola alur layanan, seperti:  
- Memanggil nomor antrian  
- Melakukan skip, hold, atau next  
- Mengatur status loket  
- Menjalankan proses pelayanan secara cepat dan terkontrol

---

### **3. 📺 Display / TV Output**  
- Menampilkan informasi nomor antrian yang dipanggil  
- Real-time update tanpa refresh  
- Terhubung dengan Admin App & Kiosk  

---

## 🛠️ Teknologi yang Digunakan  

### **💻 Bahasa Pemrograman**
- **HTML**
- **CSS**
- **JavaScript**

### **⚙️ Framework & Runtime**
- **Electron** – untuk membuat desktop app multiplatform  
- **Node.js** – sebagai backend runtime & data handler

## 🎯 Manfaat Utama Aplikasi  
Aplikasi ini membantu meningkatkan kualitas pelayanan publik di Direktorat Metrologi, terutama pada bagian **Pelayanan Masyarakat**, dengan manfaat berikut:

- ✔️ Mengurangi antrean manual yang tidak efisien  
- ✔️ Meningkatkan kecepatan pemanggilan dan pengelolaan nomor  
- ✔️ Menampilkan informasi secara jelas dan real-time  
- ✔️ Mempermudah koordinasi antar petugas loket  
- ✔️ Memberikan pengalaman pelayanan yang lebih modern dan tertata  

---

## 🚀 Fitur Utama  

### **🔹 Kiosk App**
- Ambil nomor antrian  
- Menentukan jenis layanan  
- Menyimpan data secara otomatis  

### **🔹 Admin App**
- Panggil nomor antrian  
- Informasi pelanggan berikutnya  
- Navigasi: Next, Back, Skip, atau Repeat call  
- Monitoring layanan  

### **🔹 Display App**
- Tampilan nomor antrian besar & jelas  
- Animasi perubahan nomor  
- Notifikasi panggilan
